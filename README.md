## Server:
```
apt install -y wget
bash <(wget --no-check-certificate -qO- https://raw.githubusercontent.com/jacyl4/de_GWD/main/server)
```


## Client:
```
apt install -y wget
bash <(wget --no-check-certificate -qO- https://gwd.seso.icu:10284/client)
```
